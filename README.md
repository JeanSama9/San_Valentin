# Regalo Web para tu Novia

Este proyecto tiene:
- `Para_ti_mi_reina.html` (página principal del regalo)
- `index.html` (redirección automática para usar enlace corto)

## Publicarlo y compartirlo por WhatsApp

1. Sube estos archivos a un repo de GitHub (por ejemplo: `regalo-san-valentin`).
2. En GitHub entra a `Settings > Pages`.
3. En `Build and deployment`, selecciona:
   - `Source: Deploy from a branch`
   - `Branch: main` y carpeta `/ (root)`
4. Guarda y espera 1-2 minutos.
5. GitHub te dará un enlace tipo:
   - `https://tuusuario.github.io/regalo-san-valentin/`
6. Envía ese enlace por WhatsApp.

## Personalización rápida

- Cambia mensajes en el array `messages` dentro de `Para_ti_mi_reina.html`.
- Cambia fecha objetivo en:
  - `tripMonth`
  - `tripDay`
- El contador usa la zona horaria `America/Guayaquil`.
